
# IconiK Event Atelier  

It is a pet layout project for event studio   

## Tech Stack

**Client:** Parcel, HTML/PUG, CSS/SASS, JavaScript, BEM 


## Authors

- [Anton Chertok](https://github.com/chertoha)


## Features

- adaptive layout
- otimized images
- swiper for info cards 
- drop down buttons
- customized location map
